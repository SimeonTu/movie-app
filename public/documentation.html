IMFb - A Movie Database Web App

Documentation

Introduction

IMFb is a movie database web app with its own API.
It provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request data format Response body data format
Return a list of all movies to the user /movies GET None A JSON object containing a list of all movies
Return data about a single movie to the user by title /movies/[title] GET None A JSON object holding data about a single movie by title (description, genre, director, image URL)
Return data about a genre by name /genres/[name] GET None A JSON object holding data about the name of the genre and a description
Return data about a director by name /directors/[name] GET None A JSON object holding data about a director's name, bio, birth year, and death year
Allows new users to register /users POST A JSON object containing a username, password, email, and birthday (keys should be in lowercase)

Example:
{
  "Username": "JimBob",
  "Password": "abcd1234",
  "Email": "jimbobjoe@example.com"
  "Birthday": "1945-01-01"
} 		A JSON object holding data about the user's name, password, email, ID, and favorite movies
Allows users to login /login POST A JSON object containing a username and password (Username and Password must be capitalized)

Example:
{
  "Username": "JimBob",
  "Password": "abcd1234",
} 		A JSON object containing the user's information and a JWT token, which is required as a Bearer token in the authorization section of the request headers in every subsequent request to the API
Allows users to update their info by username /users/[username] PUT A JSON object containing the information that is to be amended

Example:
{
  "Birthday": "1945-01-01"
} 		A text message confirming that the user has successfully updated the requested information
Allows users to add a movie to their list of favorites /users/[username]/movies/[movieID] POST None A text message indicating that the movie has been added to the user's list of favorites
Allows users to remove a movie from their list of favorites /users/[username]/movies/[movieID] DELETE None A text message indicating that the movie has been removed from the user's list of favorites
Allows existing users to deregister /users/[username] DELETE None A text message indicating that the user has successfully deregistered their account
